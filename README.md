이 프로젝트는 Node.js로 만든 이저도감 백엔드 API이며, PostgreSQL과 연동됩니다.

## API 주소 목록

### Flutter 에뮬레이터에서 사용

- 질문 목록 요청:  
  http://10.0.2.2:3000/api/questions

- 유형 목록 요청:  
  http://10.0.2.2:3000/api/types

- 사용자 플레이 기록 저장 (POST):  
  http://10.0.2.2:3000/api/user_play_records

### PC 브라우저 테스트용 주소 (같은 서버 기준)

- 질문 목록 요청:  
  http://localhost:3000/api/questions

- 유형 목록 요청:  
  http://localhost:3000/api/types

- 사용자 플레이 기록 저장 (POST):  
  http://localhost:3000/api/user_play_records
