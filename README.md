이 프로젝트는 Node.js로 만든 이저도감 백엔드 API이며, PostgreSQL과 연동됩니다.

## API 주소 목록

- Flutter 에뮬레이터에서 질문 요청:
  - `http://10.0.2.2:3000/api/questions`

- Flutter 에뮬레이터에서 유형 요청:
  - `http://10.0.2.2:3000/api/types`

- PC 브라우저 테스트용 주소 (같은 서버):
  - `http://localhost:3000/api/questions`
  - `http://localhost:3000/api/types`
